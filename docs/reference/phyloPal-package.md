# phyloPal: Taxonomic Color Palettes and Alluvial-Dendrogram Visualization for Microbiome Data

Tools for creating publication-ready microbiome visualizations. Provides
perceptually uniform HCL color palettes with optional hierarchical
grouping by higher taxonomy, taxonomic barplots, alluvial plots for
visualizing compositional changes across groups, and combined
alluvial-dendrogram layouts.
