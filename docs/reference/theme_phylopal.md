# A clean minimal theme for phyloPal plots

A lightweight ggplot2 theme with clean axes and no background grid,
suitable for microbiome composition plots.

## Usage

``` r
theme_phylopal(base_size = 11, base_family = "")
```

## Arguments

- base_size:

  Base font size (default: 11)

- base_family:

  Base font family (default: "")

## Value

A ggplot2 theme object

## Examples

``` r
if (FALSE) { # \dontrun{
p + theme_phylopal()
} # }
```
